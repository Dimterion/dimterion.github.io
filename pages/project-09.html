Description

A full-stack application which allows users to create accounts and track their expenses and income.

The app is built with Next.js and TypeScript. It uses Clerk for authentication and users management, Neon for database management and deployed on Vercel.

Technologies

  • - Next.js
  • - TypeScript
  • - Clerk
  • - Neon
  • - Prisma
  • - CSS
  • - Prettier (code organizing)
  • - Vercel (hosting)
