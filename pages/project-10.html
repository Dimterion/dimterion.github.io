Description

A full-stack application (blog) which shows various posts and projects. Allows to create and manage users accounts as well as to add admin accounts with additional functionality (to see statistics, manage comments, posts and users).

The app is built using MERN-stack and deployed on Render.

Technologies

  • - React JS
  • - React Router
  • - Redux
  • - JavaScript
  • - MongoDB
  • - Tailwind CSS
  • - Express
  • - Node.js
  • - Prettier (code organizing)
  • - Render (hosting)
