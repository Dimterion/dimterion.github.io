Description

This is a personal project that I work on during my free time. It's an interactive story with various choices and dialogues.

The game is built with Vite, React and deployed on Vercel. It may change in the future as I'm adjusting its look and features as I work on it.

I've also made another text adventure that continues this one's story. It can be seen here.
