Description

This app allows you to find various crypto currencies and get a list of them based on what you type in the search field. Each currency has a separate page with more information about it (like price, description, origin date, market data, etc).

The data is pulled from the CoinGecko API and updated based on it.

The app is built with React and deployed on Vercel. Most of the styling is done with Tailwind CSS.

Technologies

  • - React Router
  • - JavaScript
  • - Tailwind CSS
  • - CSS (minor adjustments for Tailwind)
  • - HTML
  • - Prettier (code organizing)
  • - Vercel (hosting)
