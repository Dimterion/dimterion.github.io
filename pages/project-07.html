Description

A fullstack application which allows users to create accounts, write notes, view their details, post updates and mark notes as complete.

The app is built using MERN-stack and deployed on Railway & Render.

Technologies

  • - React Router
  • - JavaScript
  • - MongoDB
  • - CSS
  • - Express
  • - Node.js
  • - Prettier (code organizing)
  • - Railway / Render (hosting)
