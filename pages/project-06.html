Description

I've completed the Udemy course on React and initially followed the instructions, making a similar application (which source code and links to a live demo can be seen here).

I then decided to remake it into something new and came up with an idea to create an app to keep track of the current tasks.

This app lets you add new tasks to the list, edit and delete them.

The app uses local storage to save the added tasks within the current browser upon the page refresh.

Technologies

  • - React Router
  • - JavaScript
  • - CSS
  • - HTML
  • - Prettier (code organizing)
  • - Vercel (hosting)
